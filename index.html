隱私權政策

最後更新日期：2026 年 3 月 2 日

1. 資料收集

本應用程式透過 Facebook Graph API 存取以下資料，僅用於粉絲專頁管理：

  • 粉絲專頁的貼文內容與成效數據
  • 粉絲專頁的留言與互動資料
  • 粉絲專頁的洞察報告（Insights）

2. 資料用途

所收集的資料僅用於：

  • 管理與發布粉絲專頁貼文
  • 回覆粉絲留言與訊息
  • 分析粉絲專頁成效

3. 資料儲存

所有資料儲存在應用程式管理者的本地伺服器，不會傳送至第三方服務。

4. 資料分享

我們不會將您的資料分享、出售或轉讓給任何第三方。

5. 資料刪除

如需刪除您的資料，請聯繫應用程式管理者。

6. 聯絡方式

如對本隱私權政策有任何疑問，請透過粉絲專頁私訊聯繫我們。
